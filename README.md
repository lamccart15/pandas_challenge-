PANDAS CHALLENGE – HEROES OF PYMOLI

This challenge provided the opportunity to work with Python scripting using pandas in order to analyze purchasing data for the fantasy game Heroes of Pymoli. 

The following is included in the final analysis: 

Purchasing Analysis (Total) 
- Number of unique items
- Average purchase price
- Total number of purchases
- Total revenue 

Gender Demographics
- Percentage and count of male players
- Percentage and count of female players
- Percentage and count of other/non-disclosed 

Purchasing Analysis by Gender 
- Purchase count
- Average purchase price
- Total purchase value
- Average purchase total per person

Age Demographics (broken down into bins of 4 years)
- Percentage and count of players by age

Purchasing Analysis by Age (broken down into bins of 4 years)
- Purchase count
- Average purchase price
- Total purchase value
- Average purchase total per person

Top Spenders (top 5 spenders)
- Username 
- Purchase count
- Average purchase price
- Total purchase value 

Top 5 most popular items
- Item ID
- Item name
- Purchase count
- Item price
- Total purchase value

Top 5 most profitable items 
- Item ID
- Item name
- Purchase count
- Item price
- Total purchase value
